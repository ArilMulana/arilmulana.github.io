Introduction

Candy Crush Saga, game dengan latar dunia permen adalah game yang paling laris saat ini. Game puzzle buatan King.com ini telah menarik jutaan pemain di seluruh dunia dalam waktu singkat. Game ini rilis pada 12 April 2012 di Facebook. Versi mobile App dirilis pada 14 November di tahun yang sama. Pada Maret 2013 game ini dimainkan sekitar 46 juta orang per bulan atau sekitar 1,5 juta orang per hari. Saat ini 6 juta manusia bermain game ini setiap harinya. Menurut para pengamat, game ini menjadi game yang paling banyak diunduh selama tahun 2013.

DPE Framework

Ketika saya memulai memainkan game ini menurut saya sangat simple. karena pada saat baru pertama kali main saya disuguhkan dengan 2 pilihan .

1. ) Tanpa Login 2. ) Dengan Login

Pada saat masuk game dan ingin memulainya kita akan diberikan beberapa tips / panduan yang ditampilkan secara implisit yaitu dengan cara memunculkan popup tips ketika sedang loading / masuk kedalam gamenya.

pada saat awal main kita akan melihat tampilan yg sangat memudahkan pemain awal yaitu di suguhkannya layout yang berbentuk map tetapi ada lvl di setiap map tersebut. dan selipkan beberapa alur cerita serta background map yang sesuai dengan alur cerita tersebut. sehingga pemain baru dibuat penasaran akan game tersebut.

ketika kita mengclick lvl pada game tersebut sebelum memulai nya kita diberi beberapa item bantuan yg dibuat untuk membantu player memudahkan menyelasaikan masalah pada lvl tersebut. tetapi,item tersebut memiliki quantity yaitu sesuai dengan item yg telah kita dapat setelah menyelesaikan lvl game / dari bonus didalam gamenya.

Cara bermain game ini cukup gampang, yaitu dengan menswap beberapa candy menjadi sesuai aturan yang telah dibuat lalu selesaikan misi yang ada pada lvl tersebut.

game ini dibuat tidak akan pernah selesai karena setiap user yang sudah menyelesaikan lvl pada game tersebut. akan ditambahkan lvl lagi di dalam game tersebut. dan disetiap lvl tingkat kesulitannya berbeda beda sehingga para player penasaran dan terbawa suasana dalam game ini.
